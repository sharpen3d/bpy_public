Tools and Plugins

BPx - Blender Extension Suite


The tool manager is the easiest way for me to keep [BPx] plugins up-to-date with the latest releases of Blender. Currently for best results, all [BPx] plugins should be accessed via install of the tool manager and accessed via GitHub clone. If you would like to simply try out a specific tool, you can download preview .blend files for any of the plugins, with most functionality available to you.


Download From GitHub Documentation

Emitters

This add-on utilizes a set of geometry node-groups to create animated particles, allowing for customization through a panel in the 3D Viewport. This tool makes it easy to create complex particle systems within Blender and allows for a high degree of control and no simulation baking.

Download Preview Documentation

Quick Actions

Unity FBX Export, Render to file in background, Automatic HDRI setup, Alignment tools pie menu, Easy Color Managment panel, and more! Reduce repetitive mouse clicks and menu navigation, save time with Quick Actions

Download Preview Documentation

2D Tools

2D image planes, better than ever. Load images and scale planes at exact correct pixel-scaling based on your render and camera settings. Great for logo animation, VFX, storyboarding, or bringing 2D illustrations to life.

Download Preview Documentation

File Manager

Consolidate images, fonts and sounds used in the project to a single folder. This custom panel locates, copies, and places source files into a new folder organized by data type. Great for sharing files and avoiding the inevidable dreaded pink scene.

Download Preview Documentation

Word Artist

Additional text tools and shortcuts panel, generate text with geometry node-groups allowing for shader information based on extrude depth and positioning of text, isolates front, bevel, and extrusion for seperate material settnigs. Add 'stroke' to text in a orthographic view, and more!

Download Preview Documentation

Grid Packer

Automatically render the current .blend or any image sequence to a sprite sheet. Render directly to the sprite sheet from your scene, or select a pre-rendered PNG sequence to pack into a uniform grid. Applicable as an alternative to Tsoftware like TexturePacker for simple uniform grid spacing of frames.

Download Preview Documentation
