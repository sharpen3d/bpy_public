Add-Ons &

Templates

I primarily showcase work made in Blender because of its full scriptability, integrated Python environment, and the avoidance of data collection and fees. This aligns with my belief in bringing high-quality tools to the greater artist community and completely owning the tools and work I create.

Texture Laboratory

Design micro-surface normal maps and game-ready PBR texture maps directly from Blender to engine. This template includes custom panels, shaders, and node-groups to start creating materials for props or environments for Unreal or Unity.

BPx Emitters

This add-on utilizes a set of geometry node-groups to create animated particles, allowing for customization through a panel in the 3D Viewport.

Render Controller

This template file acts as a production management tool for image sequences being rendered from multiple blend files, allowing for iteration and background re-rendering, quickly comparing and previewing sequences, and sprite-packing.

BPx 2D Tools

Load images and scale planes at exact correct pixel-scaling based on your render and camera settings.

BPx Quick Actions

Simple Actions, Tools, and Panels to reduce clicks on repetitive processes and automate redundant tasks.

BPx Word Artist

Additional text tools and shortcuts panel, allows for live shader isolation front, bevel, depth, and stroke.

BPx File Manager

Collect referenced Images, Fonts, and Sounds into a single folder. Designed for file sharing and backup management.

MacOS Terminal Launcher

Adds the ability to easily launch the blender terminal window on MacOS within the blender interface.

Research &

Development

Geometry Nodes | Python Tools | Experiments

I like to refer to these projects as 'sketches' - quickly outlining interesting technical concepts. While not being fully functional or production-ready, having a bank of these experiments is extremely useful to jumpstart R&D for needs on a project.

Donut Generator - Geometry Nodes/Python

UE5 project to bring classic minesweeper to a 3D environment and highlight real-world de-mining operations in 2024 Ukraine.

VAT Creator - Geometry Nodes/Python

Blender geometry node driven vertex animated texture source creator. Apply to any deforming mesh to generate the texture to be used for displacement in game engines.

Audio Visualizer - Geometry Nodes/Drivers

Driving aspects of geometry nodes from audio pitch and volume, extending functionality of geometry nodes with drivers.

Looping Particle Creator (now BPx Emitters)

Looping particles driven by geometry nodes in this project expanded to become the fully-functional BPx Emitters add-on.

Spritesheet Workflows

Math-based shader in blender reflecting what a flipbook shader in Unity/Unreal would look like - with variables for size, frames, start frame, looping.

Blender NDO

Now integrated into Texture Laboratory, this project was to simulate the functionality in Quixel's deprecated NDO photoshop plugin - for creating normal maps using modeled detail without baking.
