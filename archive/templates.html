About Template Files


Template files listed below are Blender files tailored for technical us in various situations, set up with internal scripts and window layouts. Templates have been designed to speed up your workflow, increase productivity, and reduce clicks.


Render Controller

Preview rendered image sequences as videos in a simple UI within the Video Sequencer. Set up batch rendering processes through a set of .blend files. Migrate sequences to new folders. Automatically run TexturePacker processes on sequences or sets of sequences without running TexturePacker GUI. Render .blend files in the background, pack resulting image sequences or any image sequence automatically in the background with TexturePacker. Quick-open any referenced .blend files. Built specifically for TexturePacker use with Unity.
